Blast Guard: The Defusal Game is a graph-based simulation project that demonstrates the application of algorithmic optimization in a constrained environment. The system models a robot responsible for diffusing multiple bombs placed across different locations.
In the final phase, the project has been extended beyond a basic simulation into a more structured and refined system. The robot operates on a graph where nodes represent locations and edges represent paths with associated costs such as distance, obstacles, and equipment benefits.
The implementation of Prim’s Minimum Spanning Tree (MST) ensures that all bomb locations are connected using the minimum possible traversal cost. Additionally, alternative paths have been introduced for comparison, highlighting the efficiency of MST-based solutions.
The system now provides consistent real-time tracking of mission parameters such as distance, time, and bomb status. Based on these constraints, the mission is evaluated as successful or failed, providing a complete simulation experience.
